# Teresita — Landing "Traslado Asistido Integral"

Landing estática, accesible y lista para publicar.

## Estructura
- `index.html` — HTML con Tailwind CDN y JSON‑LD (Organization + Service).
- `robots.txt` — indexación básica.
- `sitemap.txt` — rutas principales (ajusta con tu dominio).
- `favicon.svg` — icono simple.
- (Opcional) `og-image.jpg` — añade una imagen 1200×630 para redes.

## Personaliza
- Cambia **brand**, **teléfono**, **email** y **dominio** en este README y en `index.html`.
- Edita textos de beneficios, pasos y FAQ.
- Reemplaza `og-image.jpg` y añade tu logotipo si lo tienes.

## SEO rápido
- Título ≤ 60 y descripción 140–160 chars (ya preconfigurado).
- Mantén JSON‑LD actualizado y `sitemap.txt` con URLs absolutas.
- Añade páginas legales (Aviso, Privacidad, Cookies) si las necesitas.

## Deploy
- **Netlify**: arrastra esta carpeta y publica.
- **Vercel**: crea un proyecto estático desde este repo/carpeta.
- **GitHub Pages**: configura la rama `gh-pages` y pon los archivos en la raíz.
